# StudyHub — SDG 4: Quality Education

A web application built with Angular v19 that supports SDG 4 (Quality Education) 
by providing students with a learning resource tracker, book finder, and study 
tools powered by the Open Library API.

## SDG Chosen
**SDG 4 — Quality Education**
Target: Ensure inclusive and equitable quality education and promote lifelong 
learning opportunities for all.
